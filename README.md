## Install

```bash
npm install -g clinext
```
or

```bash
yarn global add clinext
```

## Usage

### Run
To launch the cli simply type:

```bash
clinext new
```


A full documentation is available at [https://docs.clinext.dev](https://docs.clinext.dev)
